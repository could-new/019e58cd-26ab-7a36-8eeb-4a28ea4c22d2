# Flutter Calculator

A sleek, modern, and responsive calculator application built with Flutter.

## Overview

This app provides a standard calculator interface with support for basic arithmetic operations (+, -, ×, ÷), percentages, negations, and decimal points. It is designed with a responsive layout that perfectly adapts to mobile, tablet, and desktop screens.

## Features

- **Basic Operations:** Addition, subtraction, multiplication, and division.
- **Modifiers:** Percentage calculation and number negation (+/-).
- **Deletion Controls:** Clear All (C) and backspace/delete character (⌫).
- **Responsive Design:** Optimized layout that ensures the calculator fits beautifully within its bounds, utilizing a comfortable maximum width on larger screens like desktops and tablets.
- **Cross-Platform:** Runs seamlessly on Web, Android, iOS, Windows, macOS, and Linux out of the box.

## Setup & Run

Make sure you have [Flutter](https://flutter.dev/docs/get-started/install) installed.

To run the application, open a terminal in the root directory and use:
```bash
flutter run
```

To build for web:
```bash
flutter build web
```
